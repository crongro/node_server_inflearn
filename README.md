### "Node.js 웹개발로 알아보는 백엔드 자바스크립트의 이해" 을 위한 소스코드

---

### install

```shell
git clone https://github.com/nigayo/node_server_inflearn.git
cd node_server_inflearn
npm install
nodemon app
```

브라우저를 열고 http://localhost:3000/ 접속
